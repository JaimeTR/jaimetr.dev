## Errores Comunes y Soluciones

### Error 1: No manejar estados de carga

**Problema**: La UI se congela mientras se cargan datos.

**Solución**: Implementar estados de loading:

```javascript
const [loading, setLoading] = useState(false);
const [data, setData] = useState(null);

async function cargarDatos() {
  setLoading(true);
  try {
    const resultado = await fetch('/api/data');
    setData(resultado);
  } finally {
    setLoading(false);
  }
}
```

### Error 2: Ignorar compatibilidad de navegadores

**Problema**: Código que solo funciona en Chrome.

**Solución**: Usar polyfills y feature detection.

### Error 3: No optimizar para móviles

**Problema**: La experiencia en móvil es deficiente.

**Solución**: Diseño responsive y touch-friendly desde el inicio.
